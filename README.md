# Weather-cast-ProdeskIT
A clean, responsive weather dashboard built with vanilla HTML, CSS, and JavaScript. Search any city, auto-detect your location, and get live conditions with a UI that shifts its background theme to match the weather.

Built as part of a sprint on asynchronous JavaScript and API integration — no frameworks, no build tools, just the fetch API and modern async/await.

✨ Features


Live weather data for any city, powered by the OpenWeatherMap API
Geolocation support — automatically loads weather for your current location on page load
Smart search with real-time DOM updates and no page reloads
Robust error handling — invalid cities, network failures, rate limits, and timeouts all show clear, friendly messages instead of breaking
LocalStorage caching — repeated searches within 10 minutes load instantly without hitting the API again
Dynamic theming — the background transitions smoothly between sunny, cloudy, rainy, snowy, and stormy palettes based on current conditions
Fully responsive — works on mobile, tablet, and desktop
